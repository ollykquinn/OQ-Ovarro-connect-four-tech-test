# OQ-Ovarro-connect-four-tech-test
Ovarro Software Engineer Technical Test - Oliver Quinn

I created the game, Connect Four, in Python (3.12.3). Python was selected due to its readability and ease to run on different platforms. The standard library unittest was used for testing. Two files were used, ConnectFour.py containing the game and test_ConnectFour.py containing the tests.

To run game:
Python3 ConnectFour.py

To run tests:
Python3 test_ConnectFour.py
